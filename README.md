# Splicer Templating Engine

Concatenate template files and embed/convert files

## Description

Templating engine that will replace tokens with the contents of additional
files. Tokens include directives that indicate how the incoming files should
be processed.

## Getting Started
Install it and use it.

### Dependencies

libc

### Building

1. Clone this repository
2. cd into your local cloned repository
3. make

### Installing

Place executable in your path.

### Executing program

Run the executable.

## Help

Check the help.
```
splicer -h
```
